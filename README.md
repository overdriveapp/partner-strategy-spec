# Overdrive Strategy Specification

This repository contains the standard template used by Overdrive
to collect, evaluate, and implement partner trading strategies.

Partners are invited to define their strategy logic using the
provided template. Overdrive will assess feasibility and implement
an MVP version where appropriate.
